# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
make setup       # install dependencies
make lint        # ruff check src/ tests/
make test        # pytest -q  (26 tests)
make projects    # run all 12 project scripts
make reproduce   # lint + test + projects + case-study (full pipeline)
python scripts/make_notebooks.py   # regenerate all 21 .ipynb files from source
mkdocs serve     # local docs preview (pip install mkdocs-material)
python -m pytest tests/test_metrics.py::test_mape_known_value -q   # single test
```

## Architecture

**`src/dsmanual/`** — installable package; small set of tested utilities (`mape`, `mase`, `seasonal_naive`, `standardised_mean_difference`, `clip_propensity`). `pythonpath = ["src"]` in pyproject.toml. Demonstrates the principle that shared logic lives in tested, importable code.

**`projects/pN_*/`** — 12 standalone scripts + matching `notebook.ipynb`. Scripts use `matplotlib.use("Agg")` + `plt.savefig()`; notebooks use `%matplotlib inline` + `plt.show()`. Never edit `.ipynb` directly — regenerate with `make_notebooks.py`.

**`notebooks/0N-*.ipynb`** — 8 topic notebooks generated by `scripts/make_topic_notebooks.py`. Map to `docs/0N-*.md` chapters.

**`docs/`** — MkDocs Material site. Cross-links inside `docs/case_studies/` must use `../` prefix to reach chapter files (e.g. `../07-time-series-forecasting.md`). Colab badge URL: `https://colab.research.google.com/github/lorenzoamabili/ds_manual/blob/main/PATH`.

**Windows constraint**: all `.py` files must be ASCII-only. Windows cp1252 raises `UnicodeEncodeError` on any char > 255 (em-dash, arrows, box-drawing, etc.).
