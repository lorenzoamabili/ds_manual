# Applied Data Science Manual

A structured, opinionated reference for how data science is actually practised — methods, models, best practices, pitfalls — backed by **runnable, verified Python on real open data**.

→ Jump to **[Case Studies](case_studies/index.md)** for how these techniques are applied at Netflix, PayPal, Google, Uber, DeepMind, and others.

→ **[Notebooks](notebooks.md)** — 18 Jupyter notebooks, all launchable in-browser via Binder with no local setup.

→ **[Projects](https://github.com/lorenzoamabili/ds_manual/tree/main/projects)** — 12 self-contained runnable scripts with model cards and tests.
